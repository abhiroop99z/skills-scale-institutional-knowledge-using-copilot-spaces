# OctoAcme Project Management Docs

OctoAcme uses an iterative, customer-first project management approach with clear ownership, measurable outcomes, and regular review loops. Work moves through initiation, planning, execution, release, and retrospective phases, with risk management and communication built into each stage. Core roles include Project Managers, Product Managers, Developers, QA, and stakeholders, and common artifacts include one-pagers, backlogs, release plans, and risk registers.
